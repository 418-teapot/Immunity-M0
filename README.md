# Immunity-M0

**Immunity-M0** 处理器遵循 **MIPS32 Release 1** 架构，设计类似于 **MIPS32 M14K™** 内核，采用了经典的五级流水结构。

## 环境要求

运行测试本项目需要本地机器具备Verilog仿真工具，如Vivado、Modelsim、iVerilog、Verilator等。

## 安装

`git clone https://github.com/418-teapot/Immunity-M0`
